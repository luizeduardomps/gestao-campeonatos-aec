# gestao-campeonatos-aec
Aplicação web responsiva desenvolvida para gerenciar e divulgar o 1º Campeonato de Futebol 7 da AEC. O sistema permite o cadastro de times e jogadores, além de exibir a tabela de classificação e próximos jogos. O projeto consome uma API local (JSON Server) para dados do torneio e uma API pública para a previsão do tempo nos dias de jogo.
