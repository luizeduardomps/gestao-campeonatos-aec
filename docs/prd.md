**Escopo de Páginas (Requisito Mínimo):**
A aplicação será dividida em 3 páginas principais para garantir a organização e o cumprimento dos requisitos:
1. **index.html (Dashboard/Tabela):** Página principal exibindo os dados da API (tabela de pontos) e o widget de clima da API pública.
2. **cadastro.html (Formulários):** Página dedicada à inscrição de times e jogadores, contendo validações rigorosas (Regex, HTML5), máscaras jQuery e salvamento temporário no Web Storage.
3. **regulamento.html (Sobre/Mascote):** Página focada em layout CSS puro, imagens otimizadas do mascote Lobinho da AEC e design fluido.
